创建一个 React + TypeScript 前端项目，用于混沌系统可视化。

----------------------------------------
【1】技术要求

- React
- TypeScript
- 使用函数组件
- 使用 hooks

----------------------------------------
【2】项目结构

src/
├── components/
│   ├── ModelSelector.tsx
│   ├── ParameterPanel.tsx
│   ├── PlotView.tsx
│   └── ChatPanel.tsx
│
├── pages/
│   └── MainPage.tsx
│
├── api/
│   └── client.ts

----------------------------------------
【3】基础页面

MainPage 包含：

- 模型选择
- 参数面板
- 绘图区域
- 聊天区域

----------------------------------------
【4】要求

- 不使用复杂状态管理（先用 useState）
- UI 简单即可
