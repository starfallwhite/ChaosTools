实现模型选择和参数动态生成组件。

----------------------------------------
【ModelSelector.tsx】

- 从 /models 获取模型列表
- Dropdown 选择模型
- 通知父组件

----------------------------------------
【ParameterPanel.tsx】

- 接收 param_schema
- 动态生成输入框：

规则：
- float → input
- int → input
- array → 文本输入

----------------------------------------
【要求】

- 使用 useEffect 获取数据
- 使用 useState 管理参数
- 参数变化实时更新
