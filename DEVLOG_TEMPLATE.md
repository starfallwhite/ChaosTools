# 开发日志模板

## 新日期记录模板

复制以下内容到 DEVLOG.md 的 "进度记录" 部分：

```markdown
### YYYY-MM-DD - [阶段名称]

#### 完成事项

**1. [事项名称]**
- [具体内容]
- [文件路径]

**2. [事项名称]**
- [具体内容]

#### 关键决策

| 决策点 | 选择 | 原因 |
|-------|------|------|
| [问题] | [选择] | [原因] |

#### 待解决问题

- [ ] [问题描述]
- [ ] [问题描述]

#### 代码变更

| 文件 | 变更类型 | 说明 |
|------|----------|------|
| path/to/file | 新增/修改 | 说明 |
```

---

## 阶段完成模板

当完成一个阶段时，在 "开发阶段规划" 中更新：

```markdown
### Phase X: [阶段名称] (YYYY-MM-DD ~ YYYY-MM-DD)
- [x] 已完成项1
- [x] 已完成项2
- [ ] 未完成项（原因说明）
```

---

## 里程碑模板

重大里程碑在文件开头添加：

```markdown
## 🎯 里程碑

### [里程碑名称] - YYYY-MM-DD
- [关键成果]
- [影响范围]
```
